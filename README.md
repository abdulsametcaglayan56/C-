# 🎮 C++ Bulletproof Number Guessing Game

As you know, I am currently learning C++ and wanted to share this code to track my progress. Although it might be too simple for experienced engineers, I hope it can be helpful for fellow beginners. Thanks for checking it out!

## 🚀 Key Features
* **Robust Input Validation:** If the user enters a decimal number (like `2.8`) or a character/string, `cin.clear()` and `cin.ignore()` handle the error gracefully without crashing the program. 
* **Heal (Life) System:** The user starts with 5 heals. Heals only decrease for valid but incorrect guesses.
